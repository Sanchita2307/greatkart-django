# Greatkart Django eCommerce App
Description
Greatkart is a fully-functional eCommerce web application built with Django, designed to simulate a real-world online shopping platform. Users can browse products, add them to a cart, and place orders seamlessly. The project demonstrates a strong understanding of Djangoâ€™s core features, including models, views, templates, authentication, and session management.

# Features
User Authentication: Register, login, and manage user profiles
Product Browsing: View products by category with dynamic filtering
Shopping Cart: Add, update, or remove products
Checkout and Orders: Place orders and view order history
Admin Panel: Manage products, orders, and users
Responsive Design: Works on desktop and mobile devices

# Tech Stack
Backend: Python, Django

Frontend: HTML, CSS, Bootstrap, JQery, Bootstrap
Database: MySQL
Version Control: Git & GitHub
Installation / Setup

# Clone the repository
git clone https://github.com/Sanchita2307/greatkart-django.git
cd greatkart-django

# Create and activate a virtual environment
python -m venv venv
venv\Scripts\activate   # Windows
# source venv/bin/activate  # Mac/Linux

# Apply migrations
python manage.py migrate

# Run the server
python manage.py runserver

# Open your browser and visit: 
http://127.0.0.1:8000
